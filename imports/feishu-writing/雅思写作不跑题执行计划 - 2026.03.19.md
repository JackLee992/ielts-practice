# 雅思写作不跑题执行计划 - 2026.03.19

日期：2026-03-19
核心目标：把“中文四句不跑题”练成可重复执行的固定流程
适用范围：IELTS Task 2，尤其是双问题题和优缺点题
建议周期：`4 天`
建议日时长：`50-70 分钟`

## 一、这份计划只解决一个问题

这几天不要把目标放得太大。

这份计划只解决：

`看到题目后，能不能稳定写出两段不跑题的 body。`

也就是说，这几天优先级最高的不是：

- 高级词汇
- 复杂句炫技
- 一次性写完整篇高质量作文

这几天最重要的是把下面这个流程固定住：

`审题分盒子 -> 中文四句 -> 英文四句 -> 检查有没有偏`

## 二、执行总原则

### 原则 1

`每段只回答题目里的一小问。`

双问题题不能混着答，优缺点题不能把优点缺点写成一团。

### 原则 2

`先写中文四句，再写英文四句。`

如果中文段落还不稳，英文一般只会把偏题放大。

### 原则 3

`每句只推进一步逻辑。`

固定按下面四步走：

1. 中心
2. 为什么
3. 结果
4. 例子

### 原则 4

`例子只能服务本段中心。`

如果例子更像在证明“努力”“上进”“很辛苦”，那就要停下来检查自己是不是已经偏题。

## 三、每天开始前的 3 分钟检查

每次动笔前，先快速回答这 4 个问题：

1. 这道题一共问了几个问题？
2. 我这一段到底在回答哪一个？
3. 这一段的第一句能不能直接回答题目？
4. 我的例子是不是只证明这一段的中心？

如果有两题答不清，就先别写英文。

## 四、4 天执行安排

### Day 1：把题目拆开，不许混答

目标：

`先把“审题分盒子”练熟。`

任务：

1. 选 `2` 道 Task 2 题目
2. 每题只做审题，不写全文
3. 把题目拆成小盒子
4. 每个盒子各写 `1` 句中文中心句

输出要求：

- 每题写出题目类型
- 每题写出两个 body 分别回答什么
- 每段写出 `1` 句中文中心句

完成标准：

- 看得出 Body 1 和 Body 2 没有抢同一个任务
- 每个中心句都能直接回答题目某一问

今天不要做的事：

- 不写整篇
- 不急着翻英文
- 不追求例子多

### Day 2：只练中文四句，压住跑题

目标：

`把中文段落写稳。`

任务：

1. 用昨天的 `2` 道题
2. 每题写 `2` 段中文四句
3. 每句固定承担一个功能

固定格式：

- 中心
- 为什么
- 结果
- 例子

输出要求：

- 共写 `4` 段中文四句
- 每段只保留 `1` 个中心

完成标准：

- 每段第一句都直接回答题目
- 没有一句同时承担两步以上逻辑
- 例子不会把段落带到别的方向

今天复盘重点：

1. 有没有把 `change` 偷偷写成 `effort`
2. 有没有把 `advantages/disadvantages` 写成单纯列点
3. 有没有从个人跳到社会却没有过渡

### Day 3：把中文四句翻成英文四句

目标：

`把不跑题的中文结构，稳定落成英文 body。`

任务：

1. 从 Day 2 里选 `2` 段最稳的中文四句
2. 每段翻成英文 `4` 句
3. 只做轻微升级，不做大改写

允许使用的连接方式：

- because
- therefore
- as a result
- for example
- this means that

输出要求：

- `2` 段英文 body
- 每段大约 `70-90` 词

完成标准：

- 英文段落和中文中心完全一致
- 没有因为想写复杂句而换题
- 例子仍然只服务本段中心

今天不要做的事：

- 不硬塞很高级词
- 不把整条逻辑链压进一句长句
- 不临时增加新论点

### Day 4：半套实战复测

目标：

`在接近考场的状态下，检查流程是否真的稳定。`

任务：

1. 选 `1` 道新题
2. 先花 `5` 分钟做审题分盒子
3. 再花 `10` 分钟写两段中文四句
4. 最后花 `20-25` 分钟写两段英文 body 和简短开头

输出要求：

- `1` 个清晰提纲
- `2` 段中文四句
- `2` 段英文 body
- `1` 个简单 introduction

完成标准：

- 全程没有出现明显混答
- 两段 body 分工清楚
- 至少 `80%` 的句子都在服务题目本身

## 五、每天结束后的复盘模板

每天训练后，只回答下面 5 个问题：

1. 今天哪一段最稳？为什么？
2. 今天哪一段最容易偏？偏到哪里去了？
3. 我是不是把题干关键词换成了别的概念？
4. 我的例子是在证明中心，还是在证明相关观点？
5. 明天最该继续修的，只有哪一个点？

复盘时不要泛泛地写“继续努力”。

只写具体错误，比如：

- 第二段第一句没有直接回答题目
- 例子证明的是努力，不是变化
- 优缺点只列点，没有解释

## 六、你这 4 天最该盯住的 3 个错误

### 错误 1

`题目问 change，你写成了 hard work。`

这属于相关但不对焦。

### 错误 2

`题目问 advantages and disadvantages，你只写了一串名词。`

这属于没展开，不算真正完成任务。

### 错误 3

`一段里同时想讲个人、社会、经济、心理。`

这样最容易把段落中心冲散。

## 七、这 4 天结束时的验收标准

只要你能做到下面 4 条，这个计划就算有效：

1. 看到双问题题，能先拆成两个清楚的盒子
2. 能稳定写出 `2` 段中文四句
3. 能把其中 `2` 段翻成英文而不明显偏题
4. 复盘时能说清自己到底偏在什么地方

## 八、最简执行版

如果哪天特别忙，就只做这个最低配版本：

1. 审题分盒子：`5` 分钟
2. 写 `2` 段中文四句：`15` 分钟
3. 检查是否跑题：`5` 分钟

只要这个最简版不断掉，你的训练节奏就不会散。

## 九、最后压成一句话

你现在最需要练的，不是“怎么写得更高级”，而是：

`先把每一段准确写在题目上，再考虑语言升级。`
