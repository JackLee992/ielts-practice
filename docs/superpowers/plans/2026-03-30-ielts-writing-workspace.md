# IELTS Writing Workspace Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create a lightweight repo structure for General Training IELTS writing practice and seed it with the user's current baseline plus the first guided training session.

**Architecture:** Keep all active study artifacts under `.ielts/`, keep imported historical evidence under `imports/`, and drive practice through short session notes rather than a large content library. The repo starts with tracking files and a single focused session aimed at the user's highest-leverage writing bottlenecks.

**Tech Stack:** Markdown files in the current workspace
