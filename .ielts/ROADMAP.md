## Outcome

This roadmap covers all four modules.

Writing is still the primary focus.
Speaking is now the second recovery module.
Listening is the third priority because precision, not comprehension, is still leaking marks.
Reading is the maintenance-plus-upside module.

It is not a motivation board. It is the execution route we will follow in this repo.

## Weekly Structure

Default weekly rhythm:

1. Two Task 2 cycles
2. One Task 1 cycle
3. One review/reset block

Default task ratio:

- Task 2: 2 parts
- Task 1: 1 part

Reason:

- Task 2 is currently weaker and needs more repetitions
- Task 1 still needs constant maintenance to reach Band 7 stability

## Stage 1: Build Stable Paragraphs

**Dates**: 2026-03-30 to 2026-04-06

**Main goal**:

- stop freezing when writing
- make Task 2 body paragraphs stable
- rebuild Task 1 letter body control

**Task 2 focus**:

- four-line plans
- one body paragraph
- two body paragraphs
- short introduction

**Task 1 focus**:

- complaint letter
- request letter
- apology/explanation letter
- opening and closing tone control

**Exit criteria**:

- can write two clear Task 2 body paragraphs
- can write one acceptable Task 1 response structure without major format confusion

## Stage 2: Build Full Tasks

**Dates**: 2026-04-07 to 2026-04-20

**Main goal**:

- connect paragraph control into complete tasks

**Task 2 focus**:

- full untimed essays
- double-question and advantages/disadvantages tasks
- stronger introduction and conclusion control

**Task 1 focus**:

- formal, semi-formal, and informal letters
- function control: complain, request, explain, invite, thank, suggest

**Exit criteria**:

- can complete one full untimed Task 2 with clear paragraph jobs
- can complete one full untimed Task 1 with suitable tone and task coverage

## Stage 3: Build Timed Stability

**Dates**: 2026-04-21 to 2026-05-10

**Main goal**:

- keep structure and language under time pressure

**Task 2 focus**:

- 40-minute control
- typo scan
- anti-off-topic self-check

**Task 1 focus**:

- 20-minute control
- faster planning
- stronger sentence economy

**Exit criteria**:

- can finish Task 2 near 40 minutes
- can finish Task 1 near 20 minutes
- can still spot main mistakes during self-check

## Stage 4: Build Band 7 Routine

**Dates**: 2026-05-11 to 2026-05-24

**Main goal**:

- turn skills into a repeatable exam routine

**Task 2 focus**:

- alternating prompt types
- rewrite after feedback
- repeated error elimination

**Task 1 focus**:

- faster tone selection
- cleaner action requests
- fewer grammar slips in standard sentence patterns

**Exit criteria**:

- repeated writing cycles look stable, not lucky
- fewer repeated mistakes reappear in `.ielts/ERRORS.md`
- more rules can be promoted into `.ielts/RULES.md`

## Drill Progression

Use this order unless a specific weakness forces us to step back:

1. Chinese plan
2. English paragraph
3. Two paragraphs
4. Introduction or ending
5. Full untimed task
6. Timed task
7. Timed task plus rewrite

## Review Rules

After each writing cycle:

1. Save the output
2. Identify the biggest bottleneck
3. Save repeated errors
4. Save working rules
5. Assign the next drill immediately
