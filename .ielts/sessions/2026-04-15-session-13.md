## Practical Attention Plan

The user asked for concrete methods to reduce drifting and improve attention during Listening Part 4.

### Method 1 - One-Blank Attention Anchor

Before the audio reaches the next blank:

1. keep the eyes on the current blank only
2. predict the answer type in one word:
   - noun
   - colour
   - body part
   - object
3. do not look far ahead across multiple blanks

### Method 2 - If One Blank Is Missed, Hard Reset Fast

If one answer is missed:

1. do not keep chasing the missed answer while the audio continues
2. mentally say `next blank`
3. move the pen or cursor to the next item immediately

### Method 3 - Listen For The Headword, Not Every Word

For difficult sentences, ask:

1. what is the head noun?
2. is the speaker narrowing from broad to specific?
3. is this a place, person, object, or body part?

### Method 4 - Short Retest Blocks

Do not train attention with long unfocused replays.

Instead:

1. do one 10-question set
2. review only 3 to 5 wrong items
3. stop and name the trap type
4. test the same trap again on a fresh set

### Method 5 - Spelling Is A Separate Layer

If the meaning was understood, treat the problem as spelling only.

Use a 20-second post-set spelling check for words like:

- `pregnant`
- `temperature`
