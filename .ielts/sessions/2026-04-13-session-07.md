# Session 07

## Date

2026-04-13

## Focus

Strengthen Task 2 sentence-building patterns that add one more half-step of explanation without sounding forced.

Main target:

1. do not stop at naming an advantage
2. use natural result patterns inside one clear paragraph center
3. keep the paragraph as argument, not as a list

## Pattern Bank For This Session

Use these four patterns first:

1. `..., giving employees ...`
2. `..., allowing them to ...`
3. `..., which helps ...`
4. `with ..., ...`

## Phrase Bank To Recycle

- `save commuting time`
- `greater flexibility`
- `handle family responsibilities`
- `maintain a better work-life balance`
- `make communication more direct`
- `improve teamwork`
- `face-to-face interaction`
- `quicker responses`
- `closer coordination among coworkers`
- `solve problems more quickly`

## Drill 1 - Single-Sentence Upgrade

Upgrade each bare point into a fuller sentence.

### Base points

1. Working from home saves commuting time.
2. Remote work gives people more flexibility.
3. Working in an office improves teamwork.
4. Office-based work makes communication more direct.

### Target

Turn each one into:

- point + one natural result

### Model direction

- `Working from home saves commuting time, giving employees more control over their daily schedule.`
- `Office-based work makes communication more direct, allowing teams to solve problems more quickly.`

## Drill 2 - One Center, Two Expansions

Choose one center only and write two sentences under it.

### Option A

`Working from home gives employees greater flexibility.`

### Option B

`Working in an office makes coordination more efficient.`

### Target

Sentence 1:

- center sentence

Sentence 2:

- expand with one of the four pattern shells

### Check

- both sentences must stay on the same paragraph line
- no new second center

## Drill 3 - Controlled Body Paragraph

Write one body paragraph in 90-110 words.

### Allowed structure

1. center sentence
2. one sentence with `giving / allowing / which helps / with`
3. one supporting sentence
4. one short example

### Topic choices

1. working from home is better because it saves time
2. office work is better because teamwork is faster

## Common Traps For This Session

1. do not add a new center in the last sentence
2. do not force `because / so / therefore` into every line
3. do not make the sentence long if the grammar becomes unstable
4. do not use spoken words such as `stuff`, `chill`, or `a lot better` in body paragraphs

## Self-Check Before You Stop

1. Did the paragraph stay on one center only?
2. Did at least one sentence go `point -> reason/result`?
3. Did I use one natural pattern correctly?
4. Did I keep the example short and relevant?
5. Did I reuse 2-3 natural phrases instead of translating from Chinese word by word?

## Saved Rule Candidate

Use this if today's drill feels stable:

`In each Task 2 body paragraph, add one natural result phrase such as giving, allowing, which helps, or with instead of forcing every explanation through because or so.`
