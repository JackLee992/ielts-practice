# Week 01 Writing Plan

**Week of**: 2026-03-30

## Weekly Goal

Build the first stable writing rhythm in this repo while covering both Task 2 and Task 1.

## Priority Order

1. Task 2 body control
2. Task 2 introduction control
3. Task 1 structure and tone
4. Error logging and rule review

## Planned Cycles

### Cycle 1

- Type: Task 2
- Focus: finish current double-question drill
- Output:
  - introduction
  - polished Body 1
  - polished Body 2

### Cycle 2

- Type: Task 1
- Focus: complaint or request letter
- Output:
  - brief plan
  - full 150+ word response

### Cycle 3

- Type: Task 2
- Focus: new prompt, same paragraph discipline
- Output:
  - two body paragraphs
  - one short introduction

### Review Block

- Review repeated unnatural expressions
- Review typo shortlist
- Update `.ielts/ERRORS.md`
- Update `.ielts/RULES.md`
- Update `.ielts/WEEKLY_REVIEW.md`

## Daily Minimum

If motivation is low, do not skip completely.

Use the fallback minimum:

- one Chinese four-line plan, or
- one English body paragraph, or
- one Task 1 opening + one body paragraph

## Success Standard For This Week

- complete 2 Task 2 cycles
- complete 1 Task 1 cycle
- save repeated mistakes instead of forgetting them
